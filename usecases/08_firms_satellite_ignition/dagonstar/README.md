# 08 Firms Satellite Ignition Dagonstar Workflow

This directory contains a Dagonstar-oriented workflow sketch for the use case. Dagonstar is maintained at https://github.com/dagonstar/dagonstar and should be installed/configured outside Sprtz.

`workflow.py` keeps the same step order documented in `../pipeline/README.md` and delegates execution to the self-contained demo scripts under `../demo/`. Bind real input and output paths in the workflow-engine layer before operational use.
